# /test

Run the test suite for the current project.

## Instructions

Detect the project type and run the appropriate test command:

- **Node.js/Bun**: Look for `package.json` and run `npm test`, `bun test`, or `pnpm test`
- **Python**: Look for `pytest.ini`, `pyproject.toml`, or `setup.py` and run `pytest`
- **Go**: Run `go test ./...`
- **Rust**: Run `cargo test`

If tests fail, analyze the errors and suggest fixes.
